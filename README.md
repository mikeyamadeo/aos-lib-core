# ⚠️ Deprecated — migrated to [Boostly/aos](https://github.com/Boostly/aos)

This repository has been **retired**. Its skills, prompts, and hooks now live in the **aos catalog** — **[Boostly/aos](https://github.com/Boostly/aos)** — distributed as a Claude Code plugin via the catalog marketplace.

## How to get these now

Install atoms from the catalog with the `library` skill:

```
/library list            # browse what's available
/library use <atom>      # install a skill / prompt / hook
```
